# threejs-vite — Vite + three.js boilerplate

Minimal boilerplate for a three.js scene using Vite and npm. Includes OrbitControls from three/examples.

Quick start
1. cd threejs-vite
2. npm install
3. npm run dev
4. Open http://localhost:5173/ (Vite default)

What this scaffold will include
- index.html — Vite entry that loads /src/main.js
- src/main.js — Example three.js scene (camera, renderer, simple mesh), OrbitControls, resize handling, animation loop
- package.json — scripts + dependencies
- .gitignore
